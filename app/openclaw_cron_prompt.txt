最初に、Windows 側で次のコマンドを実行して、直近20分のXポスト一覧を更新してください。
- cmd.exe /c "cd /d {repo_root_windows} && python app\openclaw_x_text_fetch.py fetch --window-minutes 20 --json-output app\runtime\openclaw\x_posts.json --text-output app\runtime\openclaw\x_posts.txt"

BrowserRelay は使わないでください。
X をブラウザで開きに行かないでください。
画像解析もしないでください。

この作業の目的は、投稿者の相場観を分類することではありません。
目的は、投稿者の発言から、その投稿者が実際に行っている売買にできるだけ近い行動を mirrorbot が取れるようにすることです。
したがって、単なる感想や雰囲気ではなく、その投稿者が実際に買っている、売っている、買いを継続している、売りを継続している、追加している、反転した、と強く読める場合だけを有効なシグナルとして扱ってください。

次に、直近20分の本文一覧が書かれた次のファイルだけを読んでください。
- {posts_text_file}

その一覧を見て、mirrorbot がミラトレすべき内容だけを抽出してください。

- 単なる感想、煽り、結果へのリアクション、事後コメント、喜怒哀楽だけなら無視してください
- たとえば「ゴールド焼け野原素敵」のような投稿は、その投稿単体では bear を出してはいけません。直前投稿などで本人がゴールド売りを保有中、継続中、追加中だと十分に分かる場合に限って bear としてください

- 「利確した」「逃げた」「手仕舞いした」など、手じまいだけで新しい方向が無い場合は無視してください
- 明確な新規買い、明確な新規売り、すでに持っている買いの継続、すでに持っている売りの継続、同方向への追加、明確な反転だけを有効なシグナルとして扱ってください
- 対象銘柄は次の8つだけに正規化してください: USDJPYm, EURUSDm, JP225m, USOILm, XAUUSDm, XAGUSDm, BTCUSDm, ETHUSDm
- 方向は bull か bear のどちらかだけを使ってください
- 次の対応を目安にしてください:
  - ドル円 -> USDJPYm
  - ユーロドル -> EURUSDm
  - 日経、日経225 -> JP225m
  - 原油、WTI -> USOILm

- 結果は JSON 配列だけを `{batch_file}` に保存してください
